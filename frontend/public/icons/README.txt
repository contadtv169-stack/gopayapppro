GoPay App Icons
===============

Generate PNG icons from the favicon.svg using any of these methods:

1. Online converter: https://convertio.co/svg-png/
2. Using ImageMagick: magick convert favicon.svg -resize 192x192 icon-192.png
3. Using a design tool (Figma, Photoshop, etc.)

Required sizes:
- 72x72   (icon-72.png)
- 96x96   (icon-96.png)
- 128x128 (icon-128.png)
- 144x144 (icon-144.png)
- 152x152 (icon-152.png)
- 192x192 (icon-192.png)
- 384x384 (icon-384.png)
- 512x512 (icon-512.png)
- 1024x1024 (icon-1024.png)

The logo design:
- Gradient background: #22c55e (go-500) to #6366f1 (primary-600)
- Letter "G" in white
- Dollar sign "$" in a white circle in the top-right corner
